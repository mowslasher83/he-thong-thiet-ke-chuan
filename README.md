# Hệ Thống Thiết Kế Chuẩn

## Giới thiệu
Hệ thống thiết kế chuẩn là một bộ công cụ nhằm hỗ trợ các nhà phát triển và nhà thiết kế trong việc tạo ra những giao diện người dùng nhất quán và dễ sử dụng. Dự án này sẽ giúp người dùng tiết kiệm thời gian trong việc tạo ra các thành phần giao diện và giúp nâng cao trải nghiệm người dùng.

## Tính năng
- **Thư viện thành phần**: Cung cấp một thư viện các thành phần UI (buttons, inputs, modals, v.v.)
- **Hướng dẫn sử dụng**: Tài liệu chi tiết hướng dẫn cách áp dụng các thành phần vào dự án
- **Tùy biến theo thương hiệu**: Cho phép tùy chỉnh các thành phần để phù hợp với thương hiệu của từng tổ chức
- **Hỗ trợ đa ngôn ngữ**: Hệ thống hỗ trợ nhiều ngôn ngữ, bao gồm cả Tiếng Việt và Tiếng Anh

## Cài đặt
Dự án có thể được cài đặt thông qua npm hoặc yarn:
```bash
npm install he-thong-thiet-ke-chuan
# hoặc
yarn add he-thong-thiet-ke-chuan
```  

## Sử dụng
```javascript
import { Button } from 'he-thong-thiet-ke-chuan';

function App() {
  return <Button label='Nhấn vào đây' />;
}
```
